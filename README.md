# 🎓 Student Grading System

A modern student grading system built using React, TypeScript, and localStorage.

## 🚀 Features

* Admin Login System
* Add Students with Subject-wise Marks
* Automatic Average Calculation
* Grade Assignment (A, B, C, D)
* Delete Student
* Data Persistence using localStorage
* Responsive UI

## 🛠️ Tech Stack

* React (TSX)
* Tailwind CSS
* Vite
* LocalStorage
